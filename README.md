# MFDEMS FreeRTOS

MFDEMS（Multi Function Desktop Environment Monitoring System），即多功能桌面环境监控系统。是之前裸机系统的升级版本，使用FreeRTOS来管理多个任务，每个任务负责不同的功能，从而更好的实现并发和控制，关于使用到的硬件部分，本项目不再赘述，详情请参考[MFDEMS裸机系统](https://github.com/141060102002/MFDEMS)。

本项目是本人学习FreeRTOS源码后的一个实验项目，旨在通过实现一个简单的桌面环境监控系统，来学习FreeRTOS的任务管理、并发编程等概念。
